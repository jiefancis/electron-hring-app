
## 📋 介绍

- 🍩 **为什么使用？** 桌面软件（办公方向、 个人工具），仍然是未来十几年PC端需求之一，提高工作效率
- 🍉 **简单：** 只需懂 JavaScript


## 如何运行

<!-- 国内镜像源 -->
npm config set electron_mirror https://npm.taobao.org/mirrors/electron/

npm install

cd frontend && npm install

npm run dev


## 打包软件

<!-- 打包win 64位 -->
npm run build-win

<!-- 打包 mac -->
npm run build-mac
